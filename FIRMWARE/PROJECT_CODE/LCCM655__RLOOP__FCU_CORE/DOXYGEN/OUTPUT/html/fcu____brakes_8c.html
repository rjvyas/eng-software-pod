Detailed Description

Brake control subsystem for the FCU.

Note
When the brake is fully deployed the airgap is 2.5 mm. The airgap increases by tan 17 deg x mm moved by lead screw. If counting revolutions, every rotation of the ball screw moves the wedge 3 mm. 0.9172 mm airgap increase per revolution.
