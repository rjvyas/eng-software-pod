Detailed Description

List of all track databases SHOULD MAKE A MULTIPLE OF 16 BYTES for better eeprom layout 8 + STRIPES(64) + SPARE(16) = 88 6 + SPARE(16) + SETPOINTS(256)*2 = 2 + = 626 16 * 40 = 640*4 = 2560/16 = 160.
